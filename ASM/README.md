# Assembler (ASM)

Tento adresář obsahuje projekty a cvičení zaměřená na nízkoúrovňové programování v jazyce Assembly (NASM), často ve spojení s jazykem C pro vstup/výstup.

## Obsah

### Algoritmy a logika
- Implementace základních logických operací a řídicích struktur.
- Podmíněné skoky, cykly a bitové operace.

### Proměnné
- Práce s registry a pamětí.
- Definice a manipulace s datovými typy (byte, word, dword).

### Práce s poli a řetězci
- Adresování paměti a iterace přes pole.
- Manipulace s textovými řetězci na nízké úrovni.

### Předávání parametrů a návratové hodnoty
- Volací konvence (calling conventions) mezi C a ASM.
- Práce se zásobníkem (stack) a předávání argumentů funkcím.
