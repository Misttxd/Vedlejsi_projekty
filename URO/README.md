# Uživatelská rozhraní (URO)

Tato složka obsahuje projekty z předmětu URO zaměřeného na tvorbu uživatelských rozhraní v různých frameworcích a technologiích.

## Obsah:
- **[Task Manager (Python)](./Task%20Manager%20(Python)/README.md)** - Aplikace pro správu úkolů vytvořená v Pythonu pomocí grafické knihovny `customtkinter`.
- V budoucnu zde přibudou podobné aplikace s grafickým uživatelským rozhraním vytvořené v dalších frameworcích (např. v **Qt** atd.).
