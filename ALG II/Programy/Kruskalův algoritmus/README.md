# Kruskalův Algoritmus

Implementace Kruskalova algoritmu pro nalezení minimální kostry grafu (MST).

## Funkce
- **Union-Find**: Používá strukturu sjednocení podle velikosti a kompresi cesty pro efektivitu.
- **Řazení hran**: Řadí hrany podle váhy pro zpracování od nejmenší.
- **Výpočet MST**: Počítá celkovou váhu minimální kostry.
