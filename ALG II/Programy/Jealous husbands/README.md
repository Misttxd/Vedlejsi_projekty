# Žárliví Manželé

Řešitel logické hádanky "Žárliví manželé" (varianta problému misionářů a kanibalů).

## Přístup
- Používá **Prohledávání do šířky (BFS)** k prozkoumání stavového prostoru.
- Hledá nejkratší sekvenci tahů pro přepravu všech párů přes řeku.
- Kontroluje platnost stavů (manželé nesmí nechat své ženy s jinými muži bez dozoru).
