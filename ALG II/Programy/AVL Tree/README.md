# AVL Strom

Implementace AVL stromu (samovyvažující se binární vyhledávací strom) v C++.

## Funkce
- **Vkládání**: Automaticky vyvažuje strom po přidání uzlu.
- **Mazání**: Odstraní uzly a znovu vyváží strom.
- **Průchody**: Podporuje Preorder, Inorder, Postorder a Level-order výpis.
- **Rotace**: Implementuje levé a pravé rotace pro vyvažování.

## Použití
Program čte sekvence pro konstrukci a mazání ze souborů předaných jako argumenty.
