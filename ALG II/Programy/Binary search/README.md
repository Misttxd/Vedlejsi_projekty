# Binární Vyhledávání

Jednoduchá implementace algoritmu binárního vyhledávání v C++.

## Funkce
- Načte datovou sadu a seznam čísel k vyhledání z textových souborů.
- Seřadí datovou sadu.
- Provede binární vyhledávání pro každé hledané číslo.
- Vypíše 'T' (True) pokud je nalezeno, 'F' (False) jinak.
