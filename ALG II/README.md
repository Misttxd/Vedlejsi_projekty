# Algoritmy II (ALG II)

Tento repozitář obsahuje pokročilejší implementace algoritmů a datových struktur probíraných předmětu ALG II v programovacím jazyce C++.

## Obsah
- **Projekt**: Soubory západajících do finálního projektu týkající se probíraných algoritmů.
- **Programy**: Cvičení a domácí úkoly z probíhající výuky a přednášek, zahrnující různé datové struktury a algoritmy.

Mezi probíraná témata zde tvořena patří:
- **Datové struktury:** AVL stromy, Binární halda (Heap).
- **Grafové algoritmy:** Kruskalův algoritmus (MST), Žárliví manželé (BFS), Analýza závislostí (DFS).
- **Ostatní:** Radix sort, Sběr mincí (Dynamické programování), Lights Out (Lineární algebra), Simplex, Binární vyhledávání.
