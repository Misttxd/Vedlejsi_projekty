# Technologie a Aplikace Mobilních Zařízení (TAMZ)

Tato složka obsahuje projekty a cvičení vytvořené v rámci předmětu TAMZ, zaměřené primárně na vývoj webových a mobilních aplikací s využitím moderních technologií.

## Obsah:
- [Terms app](./Terms%20app/README.md) - Aplikace na sledování důležitých semestrálních termínů.
- [Memory app](./Memory%20app/README.md) - Aplikace pro ukládání jednoduchých záznamů (úkolů).
- [BMI calc](./BMI%20calc/README.md) - BMI kalkulačka s historií.
- [API app](./API%20app/README.md) - Aplikace demonstrující HTTP požadavky (GET a POST) s využitím tokenu.
- [Phaser game](./Phaser%20game/README.md) - 2D arkádová hra ve Phaser 3 s tilemapou, skórováním a PWA podporou.
