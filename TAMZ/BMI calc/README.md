# BMI Calculator

Tato aplikace slouží jako jednoduchá BMI kalkulačka s možností ukládání historie výpočtů.

## Popis funkčnosti
- **Výpočet BMI**: Umožňuje zadat jméno, věk, pohlaví, výšku a váhu.
- **Hodnocení**: Na základě vypočtené hodnoty aplikace zobrazí výsledek v přehledném modálním okně (např. nadváha, normální váha, obezita).
- **Historie záznamů**: Záznamy se ukládají lokálně do paměti a uživatel k nim má přístup v záložce History, odkud je lze i odstranit.
