# Tréninkové scénáře

Tato složka obsahuje tematické scénáře pro procvičování na test z DS II.

## Jak s tím pracovat

- Nejprve si přečti `zadani.md`.
- Poté řeš úlohy v `zadani.sql`.
- Nakonec porovnej své řešení se souborem `vypracovani.sql`.

## Přehled scénářů

- `sprava-vazeb-a-xml-export/` - práce s vazebními tabulkami a export dat do XML.
- `vkladani-clanku-a-json-export/` - vložení článku přes proceduru a export do JSON.
- `dynamicke-tabulky-a-triggery/` - dynamické SQL, generování tabulek a watchdog trigger.
- `vzorkovani-a-vypocet-odmen/` - vzorkování dat a výpočet odměny za článek.
- `kontrola-konzistence-a-reporting/` - kontrola konzistence statistik a report oborů pro instituci.
