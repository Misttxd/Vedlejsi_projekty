# DS II

Tato složka je uklizená verze materiálů k předmětu Databázové systémy II.

## Struktura

- `cviceni/`
  - `cv01_sql_dotazy.sql` - SQL dotazy a úkoly z cvičení (sjednocená verze)
  - `cv02_plsql_zaklady.sql` - PL/SQL základy (deklarace, bloky, DML)
  - `cv03_procedury_funkce_triggery.sql` - procedury, funkce, triggery
  - `cv04_kurzory_a_pokrocile_ulohy.sql` - kurzory a pokročilejší úlohy
  - `cv05_odmeny_a_priprava.sql` - příprava na výpočty odměn a reporty

- `treninkove-scenare/`
  - tematické scénáře pro procvičování na test
  - každá složka obsahuje:
    - `zadani.sql`
    - `vypracovani.sql`
    - `zadani.md` (stručný popis úkolu)

## Co bylo sjednoceno

- Sjednocené názvy souborů do jednotného stylu: malými písmeny, pomlčky pro složky, smysluplné názvy podle obsahu.
- Odstraněné duplicitní kopie cvičení, ponechaná jedna hlavní verze.
- Složka `GRIND` nahrazena přehlednou částí `treninkove-scenare`.
