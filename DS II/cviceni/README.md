# Cvičení DS II

Sjednocené SQL a PL/SQL materiály z cvičení.

## Obsah

- cv01_sql_dotazy.sql - SQL dotazy a základní analytické úlohy.
- cv02_plsql_zaklady.sql - PL/SQL bloky, proměnné, výjimky a DML operace.
- cv03_procedury_funkce_triggery.sql - procedury, funkce a triggery.
- cv04_kurzory_a_pokrocile_ulohy.sql - kurzory a pokročilejší logika.
- cv05_odmeny_a_priprava.sql - příprava na test: odměny, přepočty a reporty.

## Konvence názvů

- cv01_... až cv05_... podle tematické posloupnosti.
- Jeden soubor = jedna hlavní verze cvičení bez duplicitních kopií.
