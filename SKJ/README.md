# Skriptovací jazyky (SKJ)

Tato složka obsahuje projekt z předmětu SKJ zaměřený na návrh backendové služby typu object storage (S3-like) ve FastAPI.

## Obsah:

- [Projekt - Object Storage API (FastAPI, SQLAlchemy, Pydantic, Alembic)](./Projekt/)

Projekt zahrnuje upload/download objektů, správu bucketů, účtování přenosů dat (billing) a soft delete logiku.
