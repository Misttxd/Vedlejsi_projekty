# Jazyk C

Sbírka projektů demonstrujících klíčové koncepty procedurálního programování v jazyce C.

## Obsah

### Základy a Nastavení
- **Nastavení prostředí**: Úvodní "Hello World" a konfigurace kompilátoru.
- **Cykly a podmínky**: Řízení toku programu.

### Paměť a Ukazatele
- **Ukazatele**: Práce s adresami, aritmetika ukazatelů.
- **Dynamická alokace paměti**: Správa paměti na haldě (malloc, free).

### Strukturovaná Data
- **Struktury**: Definice vlastních datových typů.
- **Práce s textem**: Manipulace s C-stringy a knihovna string.h.

### Funkce a Modularita
- Deklarace a definice funkcí, rozdělení kódu do modulů.

### Práce se Soubory
- **Textová práce se soubory**: Čtení a zápis textových dat (open, printf, scanf).
- **Projekt**: Komplexnější aplikace (např. CSV editor) spojující výše uvedené koncepty.
