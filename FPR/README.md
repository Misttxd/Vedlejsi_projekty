# Funkcionální Programování (FPR)

Tato složka obsahuje sady cvičení řešených v čistě funkcionálním jazyce **Haskell**. Soubory jsou určeny k řešení běžných problémů a algoritmů zavedením principů funkčního programovacího kódu.

## Obsah

- **AlgoritmyRazeni**: Implementace základních a pokročilých klasických řadících algoritmů.
- **GUIStruktury**: Ukázky na zavedení vlastních jednoduchých datových struktur nebo typů pro účely GUI vizualizace či zpracování.
- **MatematickeFunkce**: Řešení matematických rovnic a matematických posloupností na bázi funkčního zápisu.
- **OperaceSBludistem**: Modely hledání cest (path-finding) bludišti textové formy pomocí rekurze a seznamů.
- **TextoveOperace**: Aplikování parserů, nahrazování složitých textových struktur a analýza řetězců (strings).
- **ZákladníOperaceSeznamu**: Úvod a první kroky pro operace jako zpracování front, mapování (map), fitrování (filter) a skládání (fold).
